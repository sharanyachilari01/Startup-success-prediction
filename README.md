# Decoding Startup Success: Predicting Future Through Founder Profiles

## Overview
This project analyzes how founder profiles influence startup success and funding outcomes. Using statistical modeling techniques in R, the study explores correlations between founder attributes, startup financial indicators, and funding success.

## Objectives
- Analyze startup datasets from Crunchbase and DataLabs
- Study the relationship between founder background and startup success
- Apply regression models to predict funding outcomes
- Visualize startup trends using data visualization techniques

## Technologies Used
- R Programming
- ggplot2
- dplyr
- readxl
- caret

## Statistical Techniques Used
- Data Visualization
- Correlation Analysis
- Simple Linear Regression
- Multiple Linear Regression
- Logistic Regression

## Key Insights
- Revenue strongly influences startup profitability
- Expenses negatively impact profit margins
- Founder education has minor influence on funding success
- Startup ecosystems are concentrated in metro cities

## Dataset Sources
- Crunchbase
- Inc42 DataLabs
- LinkedIn
